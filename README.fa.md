# ابزار ریست خودکار ترافیک برای 3x-UI

ابزاری سبک و کارآمد برای ریست خودکار ترافیک **کاربران خاص** در پنل [3x-UI](https://github.com/MHSanaei/3x-ui) شما.

## معرفی

Auto-Reset-Traffic for 3x-UI یک ابزار مبتنی بر پایتون است که برای مدیران سرور طراحی شده تا بتوانند مصرف ترافیک کاربران منتخب را به‌صورت زمان‌بندی‌شده ریست کنند.
برخلاف اسکریپت‌های ریست سراسری، این پروژه فقط روی کاربرانی که در فایل تنظیمات مشخص می‌کنید عمل می‌کند و همین موضوع آن را برای سرویس‌های VPN اشتراکی، ایمن‌تر و انعطاف‌پذیرتر می‌سازد.

> این ابزار در کنار [پنل 3x-UI](https://github.com/MHSanaei/3x-ui) کار می‌کند و آن را تغییر یا جایگزین نمی‌کند — فقط برای ریست ترافیک کاربران منتخب، به آن متصل می‌شود.

## امکانات

- ریست خودکار ترافیک **فقط برای کاربران منتخب**، نه همه‌ی کاربران.
- اجرای زمان‌بندی‌شده با استفاده از **systemd timer**.
- پشتیبانی از بازه‌های زمانی روزانه، ساعتی یا سفارشی.
- منوی مدیریتی تعاملی با دستور `x3-tf`.
- پشتیبانی کامل از لاگ برای عیب‌یابی و مانیتورینگ.
- نصب و حذف با یک دستور ساده.
- پیاده‌سازی سبک با پایتون.

## پیش‌نیازها

- سرور لینوکس با `systemd`.
- پایتون ۳.
- دسترسی به پنل 3x-UI (آی‌پی/دامنه، پورت، مسیر، نام کاربری، رمز عبور).
- دسترسی root یا sudo.

## نصب

نصب‌کننده را با دستور زیر اجرا کنید:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/Mahersaber2024/Auto-Reset-traffic-3x-ui-/main/install.sh)
```

در حین نصب، اطلاعات پنل 3x-UI شما (آدرس، پورت، مسیر پایه‌ی وب، نام کاربری و رمز عبور) پرسیده می‌شود. این اطلاعات فقط برای اتصال به پنل خودتان استفاده می‌شوند.

## نحوه استفاده

پس از نصب، سرویس به‌صورت خودکار از طریق systemd timer اجرا می‌شود. برای مدیریت همه‌چیز، کافی است این دستور را اجرا کنید:

```bash
x3-tf
```

این دستور یک منوی تعاملی باز می‌کند که در آن می‌توانید:

- کاربران را به لیست ریست خودکار اضافه یا از آن حذف کنید
- بازه‌ی زمانی ریست را تغییر دهید (ساعتی، روزانه، سفارشی)
- ریست دستی انجام دهید
- لاگ‌ها را مشاهده کنید

## نحوه‌ی عملکرد

۱. اسکریپت لیست کاربران را از فایل `config.conf` می‌خواند.
۲. به پنل 3x-UI شما متصل می‌شود.
۳. ترافیک را فقط برای کاربران منتخب ریست می‌کند.
۴. این فرآیند طبق زمان‌بندی، از طریق systemd timer اجرا می‌شود.

## لاگ‌ها

تمام عملیات لاگ‌گیری می‌شوند تا بتوانید نتایج اجرا را بررسی و در صورت بروز مشکل، به‌راحتی عیب‌یابی کنید.

```bash
journalctl -u x3-tf.service -f
```

## حذف نصب

برای حذف کامل پروژه، دستور زیر را اجرا کنید:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/Mahersaber2024/Auto-Reset-traffic-3x-ui-/main/uninstall.sh)
```

## نکات مهم

- این ابزار به‌صورت پیش‌فرض ترافیک همه‌ی کاربران را ریست **نمی‌کند** — فقط کاربرانی که در `config.conf` مشخص شده‌اند.
- قبل از فعال کردن timer، از صحیح بودن ایمیل‌ها در `config.conf` مطمئن شوید.
- ابتدا روی تعداد کمی از کاربران تست کنید.
